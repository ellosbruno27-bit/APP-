# Project Summary
Este projeto é uma plataforma de gerenciamento de afiliados que fornece ferramentas para gerenciar leads, produtos e corretores. O sistema é projetado para facilitar a integração com APIs de terceiros, como Facebook e Google Ads, permitindo que os usuários configurem webhooks e realizem operações em nuvem. A solução inclui um backend robusto e uma interface frontend interativa, visando melhorar a eficiência dos negócios e a experiência do usuário. Recentemente, foram adicionadas funcionalidades para gerar QR Codes para afiliados e implementar notificações push, alertando administradores sobre novos leads, além de tornar o site responsivo e configurá-lo como uma Progressive Web App (PWA).

# Project Module Description
- **Backend**: Implementa a lógica de negócios, gerencia a autenticação e interage com o banco de dados.
- **Frontend**: Interface do usuário que permite interação com os dados e funcionalidades do sistema, agora com suporte a QR Codes, notificações push e responsividade.
- **Database**: Armazena informações sobre afiliados, produtos, leads e categorias.
- **APIs**: Integra-se com serviços externos para funcionalidades adicionais, como webhooks.

# File Description Inventory
- **backend/README.md**: Documentação do backend.
- **frontend/README.md**: Documentação do frontend.
- **requirements.txt**: Lista de dependências do projeto.
- **main.py**: Ponto de entrada da aplicação backend.
- **App.tsx**: Componente principal da aplicação frontend.
- **public/manifest.json**: Configurações para PWA.
- **public/sw.js**: Service Worker para suporte offline e notificação push.
- **src/lib/notifications.ts**: Utilitários para gerenciamento de notificações.
- **src/components/NotificationBell.tsx**: Componente para exibição de notificações.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Tailwind CSS
- **Banco de Dados**: PostgreSQL
- **Gerenciamento de Dependências**: Pip e PNPM
- **QR Code**: Biblioteca `qrcode.react` para geração de QR Codes
- **Notificações Push**: Implementação utilizando Service Workers e Push Notifications API.

# Usage
1. **Instalação de Dependências**:
   - Para o backend, execute: `pip install -r requirements.txt`
   - Para o frontend, execute: `pnpm install`
   
2. **Construir e Executar**:
   - Para o backend, execute: `python main.py`
   - Para o frontend, execute: `pnpm run dev`
